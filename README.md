# 🏥 MediConnect Smart

**AI-powered Healthcare Management Platform**

MediConnect Smart is a comprehensive healthcare application that bridges the gap between doctors, clinics, and patients through a unified mobile platform. Powered by OpenAI, it automates appointment scheduling, medicine reminders, and generates intelligent clinical summaries.

## ✨ Core Features
- **👨‍⚕️ Doctors**: Manage appointments, create digital prescriptions, chat with patients, and get AI-powered patient summaries.
- **👤 Patients**: Book appointments, set medicine reminders, chat with doctors, and track lab results.
- **🏥 Clinics**: Manage clinic information, staff, and view analytics.

---

## 🛠️ Tech Stack
- **Mobile**: Flutter 3.11+ & Riverpod
- **Backend**: Node.js, Express & TypeScript
- **Database & Auth**: Appwrite Cloud
- **AI**: OpenAI (GPT-4o-mini)

---

## 🚀 Quick Start

### Prerequisites
- Flutter SDK 3.11+
- Node.js 16+
- Appwrite Cloud account & OpenAI API key

### Setup

**1. Backend Setup**
```bash
cd backend-node
npm install
cp .env.example .env # Add your Appwrite & OpenAI keys
npm run dev
```

**2. Mobile App Setup**
```bash
cd mobile
flutter pub get
flutter run
```
