## 📥 Installation Instructions

### **Step 2: Enable Unknown Sources**

2. Go to **Security** or **Privacy**
3. Enable **"Unknown Sources"**

### **Step 3: Install**
1. Open file manager
2. Navigate to Downloads
3. Tap `mediconnect-v1.0.0.apk`
4. Tap **"Install"**

### **Step 4: Launch**

## 🐛 Troubleshooting

### **"Cannot find APK file"**
- Use the direct link: https://raw.githubusercontent.com/tatheer583/Medi-Commit-/main/releases/mediconnect-v1.0.0.apk
- Or navigate to `releases/` folder in repository

### **"Download fails"**
- Try a different download method
- Check internet connection
- Ensure sufficient storage space

### **"Installation fails"**
- Enable Unknown Sources in settings
- Clear cache: Settings → Apps → MediConnect → Storage → Clear Cache
- Uninstall previous version first

### **"App crashes"**
- Clear app cache
- Reinstall the app
- Check internet connection
- Ensure Appwrite backend is accessible
