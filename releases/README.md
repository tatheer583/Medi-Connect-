## Installation Instructions

### Prerequisites
- Android device with version 8.0 or higher
- Internet connection
- ~51 MB free storage space

### Steps to Install

1. **Download the APK**
   - Download `mediconnect-v1.0.0.apk` from this directory

2. **Enable Unknown Sources** (if not already enabled)
   - Go to Settings → Security
   - Enable "Unknown Sources" or "Install unknown apps"
   - (This option location varies by Android version)

3. **Install the APK**
   - Open your file manager
   - Navigate to the Downloads folder (or where you saved the APK)
   - Tap on `mediconnect-v1.0.0.apk`
   - Tap "Install"
   - Wait for installation to complete

4. **Launch the App**
   - Tap "Open" after installation completes
   - Or find "MediConnect" in your app drawer

5. **Initial Setup**
   - Create a new account or login
   - Select your role:
     - **Patient**: Access appointments, prescriptions, lab results
     - **Doctor**: Manage appointments, create prescriptions
     - **Clinic**: Manage clinic information
   - Complete your profile
   - Start using the app!

## Troubleshooting

### Installation Issues

**"App not installed" error**
- Ensure you have enough storage space
- Try clearing cache: Settings → Apps → MediConnect → Storage → Clear Cache
- Uninstall any previous version first

**"Unknown sources" not found**
- On Android 8+, go to Settings → Apps & notifications → Advanced → Special app access → Install unknown apps
- Select your file manager and enable it

**App crashes on startup**
- Clear app cache: Settings → Apps → MediConnect → Storage → Clear Cache
- Uninstall and reinstall the app
- Ensure you have the latest Android updates

### Runtime Issues

**Cannot login**
- Check your internet connection
- Verify your credentials
- Try resetting your password

**Features not working**
- Ensure you have internet connection
- Check if Appwrite backend is accessible
- Try restarting the app

**Notifications not working**
- Enable notifications: Settings → Apps → MediConnect → Notifications → Allow
- Check if battery saver is enabled (may block notifications)
