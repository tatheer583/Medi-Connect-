## 🐛 Troubleshooting

### **"App not installed" Error**
**Solution**:
- Ensure you have enough storage space (51 MB minimum)
- Try clearing cache: Settings → Apps → MediConnect → Storage → Clear Cache
- Uninstall any previous version first
- Restart your phone

### **"Unknown Sources" Option Not Found**
**Solution** (Android 8+):
1. Go to Settings → Apps & notifications
2. Tap "Advanced" → "Special app access"
3. Select "Install unknown apps"
4. Choose your file manager
5. Enable "Allow from this source"

### **App Crashes on Startup**
**Solution**:
- Clear app cache: Settings → Apps → MediConnect → Storage → Clear Cache
- Uninstall and reinstall the app
- Ensure you have latest Android updates
- Check internet connection

### **Cannot Login**
**Solution**:
- Verify internet connection
- Check if Appwrite backend is accessible
- Verify your credentials
- Try resetting password

### **Features Not Working**
**Solution**:
- Ensure internet connection is active
- Check if Appwrite backend is running
- Verify all permissions are granted
- Try restarting the app
